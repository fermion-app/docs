This document lists some of the frequently asked questions related to course toolings on Fermion.

<AccordionGroup>

<Accordion title="What type of courses can I create on Fermion?">
You can create a recorded course or a cohort-based course (bootcamp style). A recorded course does not have the functionality for live classes and has no fixed start or end date. Cohort-based courses have a fixed start date, end date, and include live classes inside the course.

[Read more about courses →](/setting-up-course/prerequisites#what-is-a-cohort-based-course%3F)
</Accordion>

<Accordion title="Can I bundle multiple courses together?">
Yes, you can create and sell multiple courses together on Fermion. Bundling courses allows you to sell them at a single price. You can also include other digital products like eBooks, contests, or live events in a bundle.

[Read more about course bundles →](/setting-up-course/bundle)
</Accordion>

<Accordion title="Can I sell access to my entire platform?">
Yes, you can sell "all platform access" to your students with the help of [Bundles](/setting-up-course/bundle). It's similar to a Udemy subscription, where a student can access multiple courses under the same platform for a single payment.


</Accordion>

<Accordion title="What other products can I create on Fermion?">
You can create and sell [eBooks](/ebooks/introduction), [course bundles](/setting-up-course/bundle), platform access, and more, all as part of Fermion's digital products.

</Accordion>

<Accordion title="Can I customise how my course landing page looks like?">
Yes, you can customise how your landing page looks.

[Read more about customisation →](/custom-landing-pages/introduction)
</Accordion>

<Accordion title="Does Fermion provide a landing page template or do I have to custom-code it?">
Fermion provides a default template for course and product landing pages. However, you can fully customise them if you wish.

[Learn more about custom landing pages →](/custom-landing-pages/introduction)
</Accordion>

<Accordion title="Can I enable direct checkout links for my courses?">
Yes, you can enable direct checkout links for your courses, allowing students to purchase directly.
</Accordion>

<Accordion title="Do I need to integrate any external tool to check course analytics?">
No, you do not need any external tool. Fermion shares all student data in one place.

[Read more about course analytics →](/setting-up-course/course-analytics)
</Accordion>

<Accordion title="Do I need to add custom parameters to track student performance?">
No, all student performance is tracked automatically. You can view every student’s performance under the `Students` tab inside your course.

[Learn more about student performance →](/setting-up-course/course-analytics)
</Accordion>

</AccordionGroup>
